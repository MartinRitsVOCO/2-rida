# Email Please
## Kirjeldus
See on mäng kus mängija peab ära arvama kas tegu on ehtsa kirjaga või pahatahtliku kirjaga, mis katsetab andmeid varastada või arvutit nakatada. Kirjad genereeritakse juhuslikult ettevalmistatud põhjadest.

## Tehnoloogia
NextJS + TailWindCSS
TypeScript

## Käivitamine
Tehke kloon GitHub repost ning instaleerige vajalikud paketid "npm install" käsuga. Edasi peaks olema võimalik käivitada "npm run dev" käsuga ning näha mängu kohalikus arvutis.
